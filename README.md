# Sistema-de-Estoque-com-arrays

Objetivo: Criei esse pequeno algoritmo para que eu possa melhorar meu aprendizado com arrays em Java, simulando um sistema de estoque.

Como funciona:

O programa exibe 4 opções ao usuário:

1- Cadastrar produto
2- Listar produtos
3- Mostrar valor do estoque
4- Sair

Caso o usuário digite 1, o programa pede o nome do produto a ser cadastrado, a quantidade e o valor.
Caso digite 2, o programa exibe todos os produtos em estoque e seus respectivos nomes, quantidades e valores.
Caso digite 3, o programa exibe o valor total do estoque.
Ao digitar 4, o programa se encerra.
